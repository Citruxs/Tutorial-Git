# Tutorial-Git

Este repositorio contiene un tutorial introductorio de Git orientado a explicar los conceptos y comandos esenciales para comenzar a trabajar con control de versiones.

## Contenido del proyecto

- Presentación principal en LaTeX: `Tutorial_git.tex`
- Archivos auxiliares generados durante la compilación (`.aux`, `.nav`, `.snm`, `.toc`)
- Carpeta `Imagenes/` con recursos gráficos para el tutorial

## Objetivo

Servir como material de apoyo para aprender los fundamentos de Git, incluyendo flujo básico de trabajo, seguimiento de cambios y organización de un repositorio.
